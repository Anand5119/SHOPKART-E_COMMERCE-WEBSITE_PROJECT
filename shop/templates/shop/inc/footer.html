shopkart

Shopkart is a responsive E-Commerce website developed using Django with MySQL. It is built using HTML, CSS, JavaScript, and Bootstrap. The website provides users with a list of various products available for purchase in the store. For the convenience of online shopping, a shopping cart is provided to the user. The system is implemented using Python’s web framework Django.
